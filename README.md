# Argus
Argus é a futura plataforma IoT da Universidade de São Paulo, conectando e recebendo dados de dispositivos voltados principalmente para o monitoramento da qualidade do ar.

# Passos para executar
1 - crie um arquivo .env e insira:
    MYSQL_ROOT_PASSWORD_DOCKER=sua_senha
    YSQL_PASSWORD_DOCKER=outra_senha

2 - Com o docker instalado, digite "docker compose up" no terminal
